# Tour-Guide

Tour guide is a simple website designed for helping people to find their goto places in a unknown area. It compiles all the interesting places that people can go to . It has built in search system so that people can search places. It has a chat bot integreted so that if the people have any queries they can find the answer. People can add places to their wishlist that they want to visit. Google maps has been integrated into the system so the finding location is easy. A payment gateway using stripe in implemented into the system also. The database used in the system is MySql.
